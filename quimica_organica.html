Grupos Funcionales — Química Orgánica

Todas las familias del programa. Sufijo, estructura y cómo reconocerlas.

Todos (14) Hidrocarburos (5) Oxigenados (7) Nitrogenados (2)
Hidrocarburos — solo C e H
Alcano
hidrocarburo
GrupoSolo C–C simples y C–H
Sufijo-ano
Ejemplopropano
FórmulaCH₃–CH₂–CH₃
Reconocerlo: solo C y H, todos los enlaces simples. No tiene ningún grupo especial.
Alqueno
hidrocarburo
GrupoC=C (doble enlace)
Sufijo-eno
Ejemplobut-1-eno / 1-buteno
FórmulaCH₂=CH–CH₂–CH₃
Reconocerlo: doble enlace entre dos C. El número indica dónde empieza. Si hay dos dobles enlaces → dieno.
Alquino
hidrocarburo
GrupoC≡C (triple enlace)
Sufijo-ino
Ejemplo5,5-dimetil-2-hexino
FórmulaCH₃–C≡C–CH₂–C(CH₃)₂–CH₃
Reconocerlo: triple enlace entre dos C. El número indica el C de menor numeración del triple enlace.
Aromático / benceno
hidrocarburo
GrupoAnillo de 6C, dobles alternados
Sufijobenceno
Ejemplo1,3-dimetilbenceno
Aliasmeta-dimetilbenceno (m-)
orto = 1,2 · meta = 1,3 · para = 1,4. Solo para bencenos disustituidos. Con 3+ sustituyentes usar numeración.
Halogenuro orgánico
hidrocarburo
Grupo–X (F, Cl, Br, I)
Prefijocloro- / bromo- / etc.
Ejemplo2-cloropropano
FórmulaCH₃–CHCl–CH₃
Reconocerlo: el halógeno se trata como sustituyente igual que metil. Se numera para dar el número más bajo.
Compuestos oxigenados — C, H y O
Alcohol
oxigenado
Grupo–OH en C alifático
Sufijo-ol
Ejemplo3-metilbutan-1-ol
FórmulaHOCH₂CH₂CH(CH₃)CH₃
Reconocerlo: –OH al final o en el medio de una cadena alifática. Numerar para dar el menor número al OH. Es neutro (≠ fenol).
Fenol
oxigenado
Grupo–OH en anillo aromático
Nombrefenol
Ejemplo2-clorofenol / o-clorofenol
Aliashidroxibenceno
Reconocerlo: –OH unido al anillo bencénico. El OH siempre es C1. Es ácido débil (a diferencia del alcohol que es neutro).
Éter
oxigenado
GrupoR–O–R'
Nombre[alquil A][alquil B]éter
Ejemploetilmetiléter
FórmulaCH₃–O–CH₂CH₃
Reconocerlo: O en el interior entre dos C sin doble enlace. Los grupos se nombran en orden alfabético antes de "éter".
Aldehído
oxigenado
Grupo–CHO en extremo
Sufijo-al
Ejemplo4,5-dimetilhexanal
Comúnmetanal = formaldehído
Reconocerlo: C=O en el extremo de cadena con un H (–CHO). Siempre C1. Nunca en el interior de la cadena.
Cetona
oxigenado
GrupoC=O en el interior
Sufijo-ona
Ejemplo4-metilpentan-2-ona
Comúnpropanona = acetona
Reconocerlo: C=O entre dos C (no en extremo). Numerar desde el extremo más cercano al carbonilo para dar el número menor.
Ácido carboxílico
oxigenado
Grupo–COOH en extremo
Sufijoácido ...-oico
Ejemploácido 2-etil-4-metilpentanoico
Comúnetanoico = acético (vinagre)
Reconocerlo: –COOH siempre en C1. Mayor prioridad de todos los grupos funcionales. La palabra "ácido" va antes del nombre.
Éster
oxigenado
GrupoR–COO–R' (interior)
Sufijo...-oato de ...-ilo
Ejemplopropanoato de metilo
FórmulaCH₃CH₂–COO–CH₃
Reconocerlo: –COO– en el interior. Izquierda = parte del ácido → -oato. Derecha = parte del alcohol → -ilo. Diferencia del ácido: no tiene H final.
Compuestos nitrogenados — C, H, N (y a veces O)
Amina
nitrogenado
Grupo–NH₂ / –NH– / –N–
Sufijo-amina
Ejemplobutan-2-amina
FórmulaCH₃CH(NH₂)CH₂CH₃
Primaria: –NH₂ · Secundaria: –NH– · Terciaria: –N–. Si hay sustituyente en el N → prefijo N-. Ej: N-metiletanamina.
Amida
nitrogenado
Grupo–CO–NH₂ / –CO–NHR
Sufijo-amida
EjemploN-metiletanamida
FórmulaCH₃–CO–NHCH₃
Reconocerlo: carbonilo C=O seguido de N. La cadena principal es la que incluye el C=O. Si el N tiene sustituyente → N- adelante del nombre.
Orden de prioridad — cuando hay más de un grupo funcional
Ácido carboxílico › Aldehído › Cetona › Alcohol › Alqueno › Alquino
El grupo de mayor prioridad define el sufijo del nombre. Los demás se nombran como prefijos o sustituyentes.
